# Profil fotoğrafı

Sitenin hero bölümünde fotoğrafının görünmesi için buraya **`emre.jpg`** adında bir dosya ekle.

## Nasıl eklenir (GitHub web üzerinden)
1. Şu klasöre git: `assets/img/`
2. Sağ üstte **Add file → Upload files**
3. Profil fotoğrafını sürükle, dosya adının **`emre.jpg`** olduğundan emin ol
   (farklı isimdeyse yükledikten sonra yeniden adlandır ya da `index.html` içindeki yolu güncelle).
4. **Commit changes** de.

## Öneriler
- Kare (1:1) ve net bir vesikalık/portre fotoğraf en iyi görünür.
- Yaklaşık 600×600 piksel ve 500 KB altı yeterlidir.
- Dosya yoksa site otomatik olarak "ET" monogramını gösterir; yani bozulmaz.
